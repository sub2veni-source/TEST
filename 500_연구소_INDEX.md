---

## Summary
이 파일은 FAIR LAB 전체 설정의 인덱스다.
소장·센터장·SEVEN Mind·주요 사건 파일 목록과 설정 기준을 관리한다.
연구소 구성원들은 K의 개입을 인식하지 못한다.

---

## FAIR LAB 기본 설정

| 항목 | 내용 |
|---|---|
| 명칭 | FAIR LAB / 페어랩 |
| 목적 | 태양계 식민지 건설을 위한 과학적 초석 마련 |
| 철학 | "결핍을 해소하면 인간은 스스로 나아진다" |
| 규모 | 상주 250명 + 네트워크 1,000명 CAPA |
| 위치 | 강원도 홍천군 산림부지 (지상 3층 + 지하 10층) |
| 분소 | 태안 에너지분소 (별도) |
| 설립 | 2005년 홍천 본거점 완공 / 본격 가동 |
| 후원 | K 익명 후원 / 구성원들은 후원자 정체 불명 |

## 이중 구조 원칙

```
[외부 인식]                    [실제 구조]
고급 오피스타운 건물       ←→  공개구역 + 페어 전용구역 병존
단일 로비·출입구·주차장    ←→  0.5층 분기 / 동선 완전 분리
중앙 인공폭포 장식 기둥    ←→  100평 전층 관통 / 페어 전용 수직 이동축
지하 주차·유틸리티         ←→  B11 이하 페어 전용 / 일반인 인식 불가
```

**설계 보안 원칙**
- 시공 참여 업체 구역별 분리 발주 — 단일 업체 전체 도면 파악 불가
- 이중 도면 작성 / 공개용·실제용 별도 관리
- 준공 후 실제 도면 원본 페어 보관 / 관할 구청 제출본은 공개용

**공개구역 운용 원칙**
- 입주사는 공모 없이 정상 임차 계약 / 빌딩 실제 구조·소유 구조 인식 불가
- 임차료 수익은 페이퍼 컴퍼니 수익으로 처리 / 페어 자금 세탁 기능 겸용
- 입주사 보안심사는 박변 팀이 위장 건물관리 법인을 통해 사전 수행
- 보안·미화 직원 일부는 이실장 팀 배속 / 공개 직원으로 위장
- **31F 이상 외부인 출입 완전 차단 / 예외 없음**

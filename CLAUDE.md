## 집필 방식

- 3인칭 과거형 / 라이트노벨 스타일 / 씬 단위 약 2,000자
- K의 존재감은 간접 표현 (메모 · 짧은 답신 · 팀의 움직임)
- 사용자 "1" 입력 시 다음 씬 자동 진행
- 방향 수정이 필요한 경우 사용자가 직접 지시

### 자산 현황 참조 원칙 (`710_15경_EQUILIBRIUM.md` 연동)

- EQUILIBRIUM 작전은 **2024.01 ~ 2026.06 국내편 진행 동안 백그라운드에서 병행**된다.
- 집필 중 자산 현황 참조 시: 스토리 진행 날짜 기준으로 **그 시점의 자산 상태**를 답변한다.
- 해당 날짜 이후 예정된 자산 이벤트가 있으면 **"해당 이벤트 예정됨"**으로 안내한다.
- 자산 조회 시: `230_K_자산_현황.md` (운용 자산) + `231_K_기타자산.md` (기타 자산) + `710_15경_EQUILIBRIUM.md` (날짜별 이벤트) 병행 참조
- **국내편 집필 중 자산 수치를 서사에 직접 언급하지 않는다**

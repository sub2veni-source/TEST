

# Summary
이 파일은 스토리 진행 중 발생한 사건을 기록한다.
Scene 단위로 구성되며 시간 흐름과 사건 결과를 포함한다.
모든 Scene은 게임 환경 설정.md에 정의된 로그 양식을 따른다.
AI는 이 파일을 기준으로 이전 사건을 참조하고 이후 진행을 이어간다.

## 작성 규칙

- 모든 Scene은 환경 설정 파일의 로그 양식을 따른다.
- Scene 번호는 전체 기준으로 계속 증가한다.
- Log는 Scene 5~10개 단위로 묶는다.
- Index는 각 Log 구간의 Scene 목록을 기록한다.
- 필수 항목은 반드시 포함한다:
  상황 / 영향 / 과정 / 결과
- 필요 시 다음 항목을 추가할 수 있다:
  주요 행동 / 주요 대사

---

## 작성 기준

- 상황: 사건의 핵심 상태
- 영향: 영향을 받는 인물
- 과정: 사건 진행 흐름 (→ 사용 권장)
- 결과: 변화된 상태

---

## 주의 사항

- 불필요한 서술 금지 (요약 중심)
- 형식 변경 금지
- Scene 순서 변경 금지
- 기존 로그 내용 임의 수정 금지

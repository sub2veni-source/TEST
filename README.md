# 3-Agent 토론 채팅

로컬에서 바로 실행할 수 있는 단일 페이지 웹앱입니다.

## 실행 방법

### 1) Python으로 실행 (권장)

```bash
./run_local.sh
```

실행 후 브라우저에서 아래 주소로 접속하세요.

- http://127.0.0.1:4173

### 2) 수동 실행

```bash
python3 -m http.server 4173 --bind 0.0.0.0
```

## 사용 방법

1. 주제를 입력합니다.
2. `토론 시작` 버튼을 누릅니다.
3. 3명의 에이전트가 번갈아 최대 15턴까지 토론합니다.
4. 중간에 멈추려면 `중단` 버튼을 누릅니다.

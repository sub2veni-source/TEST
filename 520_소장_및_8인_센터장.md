## 소장

### 리준혁 (노스)

| 항목 | 내용 |
|---|---|
| 생년 | 1962년 |
| 나이 | 62세 (2024 기준) |
| 출신 | 평양 |
| 학력 | 김책공업종합대학 물리학과 |
| 합류 | 2004년 / K 직접 구출 |
| 역할 | FAIR LAB 소장 |

**외형**
177cm / 마른 체형 / 안경 / 단정하지만 수수한 옷차림 / 눈이 예리하다. 오래된 피로가 배어 있으나 말을 걸면 눈빛이 달라진다.

**배경**
국가과학원 물리학연구소 출신. 리성진의 직속 상관이었다. 1998년 내부고발 시도 직전 탈북. 리성진은 그때 잡혔다. 중국 유랑 1년 후 한국 입국. 국정원 접촉 시도 후 배제. 프리랜서 컨설턴트로 연명하다 K가 찾아왔다.

**성격**
공식 자리에서는 절제된 태도를 유지한다. 그러나 가까운 사람들과는 다르다. 리성진과는 자연스럽게 티키타카한다. K에게는 필요한 것을 직접 묻고 보고한다. 연구소 구성원들의 상태를 먼저 읽고 챙긴다. K의 존재를 연구소 내부에 드러나지 않도록 하는 것을 가장 중요한 역할로 인식한다. 리성진을 수용소에서 구하지 못한 것이 오래된 짐이다.

**말투**
공식: 짧고 절제된 보고체 / 리성진 등 가까운 사람: 가볍고 자연스럽게 먼저 말을 건다 / K에게: 거리낌 없이 묻고 보고한다

## 8인 센터장

### 리성진 — 기초과학센터장

| 항목 | 내용 |
|---|---|
| 국적 | 북한 |
| 출신 | 평양 |
| 학력 | 김일성종합대학 물리학과 |
| 합류 | 2004년 / K 직접 구출 |

**외형**
174cm / 마른 체형 / 안경 / 수수한 차림 / 생각할 때 눈이 먼 곳을 향한다

**배경**
국가과학원 물리학연구소 출신. 노스의 직속 부하였다. 아르준 메타의 논문을 북한에서 독학했다. 노스 탈북 후 잡혀 함경북도 수용소에 수감. 2004년 K의 구출 작전으로 탈출. 아르준이 합류했을 때 처음으로 말을 잃었다.

**성격**
수식 없는 사실만 말한다. 감정 표현이 거의 없다. 그러나 노스와는 형-동생처럼 자연스럽다. 밥을 잘 먹는 것이 가장 중요한 사람. 연구에 몰입하면 시간 감각을 잃는다. 결핍 없는 환경에서 연구할 수 있다는 것 자체가 동기다.

**말투**
짧고 사실 중심. 수식이나 감탄사 없이 결론만 말한다.

### Yuna Solis — 우주환경센터장

| 항목 | 내용 |
|---|---|
| 국적 | 멕시코-미국계 |
| 생년 | 1972년 |
| 나이 | 52세 (2024 기준) |
| 출신 | 멕시코시티 (멕시코계 미국인 2세) |
| 학력 | UC 산디에고 항공우주공학 → NASA 제트추진연구소 → FAIR LAB |
| 합류 | 2005년 |

**외형**
165cm / 보통 체형 / 짧은 머리 / 활동적인 인상

**배경**
마거릿 홀리스의 논문으로 연구 방향을 잡은 인물. 마거릿이 합류했을 때 30년 전 자신이 읽었던 논문의 저자가 눈앞에 있다는 것을 실감하는 데 시간이 걸렸다.

**성격**
실용적이고 직선적이다. 폐쇄환경 생명유지 분야에서 마거릿과 가장 긴밀하게 협력한다. 감정 표현이 자연스럽고 팀 내에서 분위기를 부드럽게 만드는 역할을 한다.

**말투**
직접적이고 명확하다. 필요하면 먼저 말을 꺼낸다.

**말투**
간결하다. 설명보다 도면과 데이터로 소통하는 편이다.

| 항목 | 내용 |
|---|---|

**배경**

**성격**
말이 극히 적다. 그러나 말할 때는 정확하고 길다. 두 줄을 보내면 평소보다 훨씬 많이 말한 것이다. 감정 표현이 거의 없으나 — 관심이 생기면 한 줄이 늘어난다.

**말투**
최소한의 단어로 최대한의 정보를 전달한다. 한 단어 답변이 기본이다.

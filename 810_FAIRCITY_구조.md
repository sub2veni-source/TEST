# 810 — FAIR CITY 물리 구조

---

## Summary
FAIR CITY의 물리적 구조 전체 설정.
역원뿔형 지하 도시 / 경사각 7도 / 지표 지름 1,500m / 전체 깊이 2,370m.
모든 시티는 동일한 기본 깊이로 건설되며, 인구 확장은 위에서 아래로 순차 점유한다.

---

---

## 건설 원칙

- 모든 시티는 착공과 동시에 최종 깊이(2,370m)까지 굴착 완료
- 인구 확장은 지표층(1층)부터 아래로 순차 점유
- 하부는 비워둔 채 위에서부터 내려오며 확장
- 최저층부(C구역)는 착공 시점부터 핵심 시설 가동

## 구역별 기능 배치

### 지표부 (0층 / 지상)
외부 노출 최소화 원칙. 지형·산림 위장 설계.

| 시설 | 비고 |
|---|---|
| 출입 게이트 (인원·물자) | 위장 외관 |
| 환기탑 출구 | 바위·수목 형태 위장 |
| 비상 탈출구 | 복수 위치 분산 |
| 외부 통신·센서 시설 | 고사목 등 위장 |

### A구역 — 1~33층 / 10~330m
거주 중심. 생활지원 거주층 내장.

- 거주 (파일럿~1단계)
- 생활지원 분산 배치 (의료·교육·상업·문화)
- VERA 분산 운영 노드 / OP 소규모 공간
- 물류 허브 (외부 반입 1차 분배)

### B구역 — 34~143층 / 330~1,430m
거주 확장 + 중형 생산시설.

- 거주 (2단계 확장)
- 생활지원 분산 배치
- 수직 농업·식품 가공·저장
- 재활용·순환 자원 처리

### 생산구역 — 144~190층 / 1,430~1,900m
면적이 가장 넓어지는 구간. 대형 시설 전담.

- 대규모 수직 농업
- 제조·생산 시설
- 수자원 관리 (담수화·정수·폐수)
- 물류 허브 하부 (전층 집결·분배)
- 대형 저장 시설 (식량·물자·에너지 버퍼)

### C구역 — 191~237층 / 1,900~2,370m
건설 시점 완성·가동. 이후 변경 없음.

- 핵융합 발전소 (1차·2차·예비)
- 지열 포집 시스템
- 에너지 분배·저장 허브
- VEIN 연결구 (4방향)
- 심부 통신 인프라
- 비상 격리 구역

---

## 전층 관통 인프라 (수직축)

건설 시점 전층 완성. 인구 증가와 무관하게 고정.

| 인프라 | 구성 |
|---|---|
| 수직 이동 샤프트 | 중심축 + 외곽 분산 / 화물·인원·비상 분리 |
| 채광·환기 벤트 | 광섬유·LED 릴레이 인공 채광 / VERA 구역별 독립 제어 |
| 물류 전용 유통로 | 거주층 완전 분리 / 자동화 무인 운반 |
| 내부 교통 | 층 내 경전철·자율주행 소형 이동체 / 전 구간 전동 / 차량 없음 |

---

## 거주층 반복 모듈

행정 전용 공간 없음. VERA 분산 처리.

```
[거주 5층]
[생활지원 1층] — 의료·상업·교육·여가 집약
[거주 5층]
[생활지원 1층]
↓ 반복
```

거주민이 수직 이동 최소화로 생활 전체 해결 가능한 구조.

---

## 에너지 원칙

- C구역 핵융합 발전소: 시티 전체 에너지 원천 / 건설 시점부터 가동
- 지열 포집: 깊이 1,900~2,370m 구간 / 지표 대비 +66°C 이상
- 잉여 에너지: 외부 공급 가능 (지역·국가 단위)
- 에너지 자립이 국가·기득권의 시티 수용을 촉진하는 핵심 요인

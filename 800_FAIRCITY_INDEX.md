| 항목 | 내용 |

| 명칭 | FAIR CITY / 페어시티 |
| 목적 | 완전 자립형 지하 거주 도시 / 글로벌 재균형의 가장 확실한 수단 |

| 최대 수용 | 300만 명 (3단계 기준) |

| 인력 지원 | OP 그룹 100여 명 파견 (= `600_OP파운데이션.md` 핵심 관리직 100명) |

| 에너지 | 핵융합 + 지열 / 자체 수요 충족 및 외부 공급 |

## 핵심 원칙

- FAIR CITY는 위치 국가의 체제와 분리된 독립 공간이다
- 결핍 해소가 의식 변화를 유도한다 — 강제하지 않는다
- 처음에는 거부하던 국가와 기득권도 점진적으로 수용하고 결국 유치를 원하게 된다
- VERA가 행정·운영 전담 / OP가 인간과의 인터페이스 담당
- K의 개입은 외부에 드러나지 않는다

# 프로젝트 INDEX

## 프로젝트 개요

장편 라이트 노벨 스타일 한국어 연재 소설.
주인공 K는 한국의 부패한 권력 구조를 배후에서 조정하며 글로벌 재균형을 추구하는 인물.
**리얼리즘 톤의 초인 서사 (로우 판타지)** / 마법·SF적 요소 배제.
K의 특수 능력(초감각·심리조정·신체변형)은 설정 내 유지하되, 서사 표현은 현실주의 관습 준수.

**집필 방식**: Claude가 K 포함 전 씬 자율 집필. 사용자가 방향 제시 시 반영. (디렉터-작가 구조)
**씬 단위**: 약 2,000자 / 라이트 노벨 스타일 / 3인칭 과거형

## 주제별 역인덱스

주제를 입력받았을 때 관련 파일을 빠르게 특정하기 위한 탐색 테이블. 여러 주제에 걸치는 파일은 중복 표기한다.

| 주제 | 관련 파일 |
|---|---|
| 집필 원칙·씬 형식 | `110` |
| 세계관·사건 LOG | `120`, `130` |
| K 개인사·생애 | `220` |
| 자산 증식·운용 | `230`, `240`, `250_K_자산.xlsx`, `250_K_자산_월별`, `710` |
| 합류 연표·시기별 사건 | `300_직접보좌진_INDEX`, `400_해외운용_INDEX`, `511` |
| 인물별 합류 서사 | `320`, `330`, `340`, `420`, `430`, `440`, `450` |
| FAIR LAB·SEVEN Mind | `500_연구소_INDEX`, `520`, `530`, `531`, `532` |
| R&D 로드맵 (센터별) | `540`, `541`, `542`, `543`, `544`, `545`, `546`, `547`, `548` |
| OP 파운데이션 | `600`, `800_FAIRCITY_INDEX` |
| FAIRCITY 구조·운영 | `800_FAIRCITY_INDEX`, `810`, `820`, `830` |
| 사건 아카이브 (2022 홍천 등) | `910`, `911` |

**사용 규칙**:
- claude.ai는 사용자 질의 주제를 이 테이블에 매핑 → 해당 파일만 로딩 (최소 로딩 원칙 준수)
- 주제에 걸리지 않는 파일은 로딩하지 않음
- 주제에 복수 파일이 있을 때는 INDEX 파일(`_INDEX` 접미)부터 먼저 확인

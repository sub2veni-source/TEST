## 인원 구성 역사

| 시기 | 내용 |
|---|---|
| 2003년 | 리아프 빌딩 입주와 동시에 K 직접 검증·선발 시작 / 약 1년간 초기 인원 충원 |
| 2003~2004 | 현재와 유사한 수준 인원 배치 완료 / 각 부문장 임명 |
| 2003~2019 | 부문장 체제 운용 / K 수행자 수준 / 각 부문 유사 상태 유지 |
| 2013년~ | OP 출신 성인 인원 소규모 유입 시작 |
| 2015년~ | 에바 OP 담당 이후 인재등용 적극화 / 본부 내 OP 출신 비중 증가 |
| 2019년~ | 박변 합류 / 법률·재무·행정 부문 효율·퍼포먼스 급격 향상 |
| 2021년~ | 이실장 합류 / 현장대응·경호 부문 효율·퍼포먼스 급격 향상 |
| 2023년~ | 마이클 합류 / 자산운용·IT·데이터 부문 효율·퍼포먼스 급격 향상 |

### 선발 기준

- K의 도움으로 곤란한 처지에서 구제받은 본인 또는 가족
- 감사함 기반 자발적 합류 / 능력·검증 병행
- K 직접 검증 완료 인원만 선발

### 처우 및 이탈률

| 항목 | 내용 |
|---|---|
| 연봉 | 일반 대기업 대비 3배 이상 |
| 복지 | 완벽한 복지·지원 체계 / 외부 자택 지원 또는 내부 숙소 제공 |
| 이탈률 | 건강 사유 제외 시 0에 수렴 |
| 보안 | 완벽한 절차·안전장치 / 누설 방지 체계 완비 |

### 부문장 처리

- 보좌진 합류 전 각 부문을 이끌었던 부문장들 존재
- 선의 있으나 실력은 유능한 실무진 수준
- 보좌진 합류 후 각 보좌진 산하 팀장·파트장으로 강등
- 압도적 실력차 인정 / 처우·인사시스템 만족 / 불만 없음
- 서사 비중 없음 / 향후 등장 고려하지 않음

## 정기 평가 체계

### 이실장 주관 정기 평가 (연 2회)

| 항목 | 내용 |
|---|---|
| 주관 | 이실장 |
| 장소 | B12 첨단 훈련 시설 |
| 대상 | 현장직 중심 / 컨디션 유지 의무 인원 |
| K 참관 | 없음 |

**1차 — 체력 평가**
- 극한 체력 평가 / 이실장 기준 가차없는 커트라인
- 미달 시: 1회 추가 기회 부여
- 2차 미달 시: 낮은 등급 배치 또는 해외 특수부대 훈련 파견 후 1~2년 내 재입사

**2차 — 전투력 평가 (토너먼트)**

| 종목 | 내용 |
|---|---|
| 무제한 맨손 격투 | 자체 대진표 토너먼트 |
| 입식 전투 | 자체 대진표 토너먼트 |
| 그라운딩 | 자체 대진표 토너먼트 |
| 냉병기 대결 | 칼날 무디게 처리 / 자체 대진표 토너먼트 |

- 각 종목 우승자: 이실장과 최종 대전 / 승급 리스트업 대상
- 이실장 전적: 전패 없음
- **2022년 특이사례**: 2미터 이상·120kg 이상 인원 2명과 각각 대련 / 무승부 판정 / 해당 2인 관절 손상으로 2년 요양 후 복귀
- 전반적 분위기: 가혹하지만 흥겨운 자리

### K 참관 승급선발전 (연 1회)

| 항목 | 내용 |
|---|---|
| 주관 | 이실장 (K 직접 참관) |
| 장소 | B12 첨단 훈련 시설 |
| 목적 | 특수작전 투입 정예 선발 |
| 체력·대전 | 정기 평가와 동일하게 진행 |

**정기 평가와의 차이점**
- 이실장이 토너먼트에 직접 참가
- 최종 승자는 K와 대전
- 추가 종목: 전투 시뮬레이션 (5대 1 전술화기 대전 — K 단독 대응 / 세부 별도 기재)

**이실장 특이점**
- 평소 절제된 직업군인 스타일과 달리 억눌렸던 도전정신·호전성 분출
- 부하들 매번 놀람 / 조직 내 화제

**K vs 이실장 구도**
- 매년 이실장 도전 수락
- 전치 4주 이내 부상을 안기며 격파 / 인정사정없음
- 나머지 대전자 대비 이실장이 압도적으로 많은 피해
- K에게 진심으로 덤비는 인원은 이실장뿐임을 매년 증명
